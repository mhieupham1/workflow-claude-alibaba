Web Search Command - Search the web for information using Gemini CLI.

When invoked, execute the following steps:

## Step 1: Understand the search query
- Identify what information the user is looking for
- Clarify the search scope if the query is ambiguous

## Step 2: Use Gemini CLI to search
- Execute web search using Gemini CLI
- Example: `gemini search "<query>"` or use appropriate Gemini CLI command
- Search for relevant, up-to-date information

## Step 3: Process and summarize results
- Filter out irrelevant information
- Summarize key findings from search results
- Include source links when available

## Step 4: Present results to user
- Display summarized information in a clear, organized format
- List key takeaways
- Provide links to original sources for further reading

---

**Note:** This skill uses Gemini CLI for web search functionality. Ensure Gemini CLI is installed and configured before using this command.
